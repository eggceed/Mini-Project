# Mini-Project

# Smart Home Lighting Hardware

## Requirements
* C++11-compatible compiler (clang, gcc etc.)
* PlatformIO Core

## How To's
### Install Depedencies
```
pio pkg install
``` 

### Compile and Build
```
pio run
```

### Upload to Board
```
pio run -t Upload
```

### Serial Monitor
```
pio device monitor
```
